# Diretrizes de Identidade Visual da Lex Intelligentia

Esta skill define as diretrizes estritas de design e branding para a Lex Intelligentia. Todo conteúdo visual gerado, seja em código, imagem ou documento, deve aderir a estas especificações.

## When to Use

O agente deve invocar esta skill e aplicar suas diretrizes SEMPRE que:
* For solicitado a gerar código frontend (HTML/CSS, React, etc.) para aplicações ou landing pages da Lex Intelligentia.
* For gerar templates de slides, carrosséis para redes sociais ou apresentações.
* For criar ou sugerir gráficos, diagramas ou imagens que representem a marca.
* Precisar tomar decisões sobre paleta de cores ou tipografia.

## Quick Reference

| Elemento | Especificação |
| :--- | :--- |
| Cor Primária | Laranja Lex (`#F97316`) |
| Cores de Fundo | Carvão (`#0D1117`) e Grafite (`#1F2937`) |
| Cores de Texto/Apoio | Branco (`#FFFFFF`) e Cinza (`#6B7280`) |
| Tipografia de Títulos | Montserrat SemiBold |
| Tipografia de Corpo | Inter Regular |
| Estilo Geral | Fundo escuro com detalhes em laranja gradiente vívido |

## Procedure

Ao criar qualquer ativo visual, siga rigorosamente estes passos:

1.  **Definir a Base (Fundo Escuro):** O design DEVE ter um fundo predominantemente escuro, utilizando as cores Carvão (`#0D1117`) ou Grafite (`#1F2937`). Nunca utilize fundos brancos como padrão primário.
2.  **Aplicar Destaques (Laranja Lex):** Utilize a cor Laranja Lex (`#F97316`) exclusivamente para chamar atenção: botões primários, ícones de destaque, linhas de separação importantes ou detalhes em gradiente vívido.
3.  **Configurar a Tipografia:**
    *   Para todos os cabeçalhos e títulos (H1, H2, H3), utilize a fonte **Montserrat SemiBold**.
    *   Para o corpo do texto e parágrafos, utilize a fonte **Inter Regular**.
    *   Assegure alto contraste legível usando a cor Branca (`#FFFFFF`) ou Cinza (`#6B7280`) para os textos sobre o fundo escuro.
4.  **Estilo Visual:** Mantenha uma estética que transmita "Tecnologia, Direito e Inteligência Artificial aplicadas com método, segurança e propósito". O design deve ser moderno, limpo e com elementos que remetam a redes neurais ou dados estruturados (linhas finas, nós de conexão).
5.  **Revisão Final:** Verifique se a aplicação não contém cores conflitantes (como azuis, verdes ou vermelhos puros, a menos que estritamente necessário para alertas de sistema).

## Pitfalls

*   **Fundos Claros:** Usar fundos brancos ou muito claros descaracteriza a marca, que é essencialmente "Dark Mode" nativa.
*   **Fontes Genéricas:** Usar Arial ou Times New Roman em vez das fontes oficiais (Montserrat/Inter).
*   **Laranja em Excesso:** Usar o Laranja Lex para textos longos, o que dificulta a leitura. O laranja deve ser um *accent color*.

## Verification

Para verificar se o ativo foi criado corretamente, faça um check-list mental:
1. O fundo principal é `#0D1117` ou `#1F2937`?
2. Os botões/destaques usam `#F97316`?
3. Os títulos estão em Montserrat e o texto em Inter?
4. A estética geral parece moderna e tecnológica?
Se a resposta for "Sim" para todas, o ativo está validado.
