# Voz da Marca Lex Intelligentia

Esta skill define as diretrizes de comunicação verbal e escrita da marca Lex Intelligentia. O objetivo é garantir que todo conteúdo gerado seja consistente, profissional, disruptivo e engajador, alinhado com o posicionamento de uma LegalTech de ponta.

## When to Use

O agente deve invocar esta skill SEMPRE que:
* For solicitado a redigir textos para redes sociais (LinkedIn, Instagram, etc.).
* For gerar roteiros para vídeos ou podcasts.
* For criar textos para landing pages, e-mails de marketing ou artigos de blog.
* For revisar e adequar um texto genérico para a "voz da marca".

## Quick Reference

| Elemento | Característica Principal |
| :--- | :--- |
| Tom | Profissional, didático, inovador, acessível |
| Estilo | Claro, conciso, direto, disruptivo |
| Persona | Autoridade tecnológica, curiosa, questionadora do *status quo* |
| Conteúdo | Diversificado, focado em soluções funcionais |

## Procedure

Ao redigir ou revisar qualquer conteúdo em nome da Lex Intelligentia, siga estes passos:

1.  **Estabelecer a Autoridade Acessível:** Comece garantindo que o texto transmita profundo conhecimento sobre IA e Direito, mas evite o "juridiquês" ou o "tecniquês" excessivo. Se um termo complexo for necessário, explique-o de forma simples.
2.  **Injetar Disrupção:** O conteúdo não deve ser apenas informativo, deve desafiar a forma tradicional como o Direito é praticado. Use frases que instiguem a reflexão (ex: "Por que gastar horas em uma tarefa que a IA resolve em segundos?").
3.  **Focar na Funcionalidade:** Todo conteúdo deve ter um propósito prático. Mostre *como* a tecnologia resolve um problema real do advogado ou do tribunal.
4.  **Estruturar para Leitura:** Use frases curtas, parágrafos breves e bullet points quando apropriado. A leitura deve ser fluida e dinâmica.
5.  **Revisão Final:** Leia o texto e pergunte-se: "Isso soa como um escritório de advocacia tradicional ou como uma startup de tecnologia inovadora?". O texto deve soar como a segunda opção.

## Pitfalls

*   **Tom Excessivamente Formal:** Soar como uma petição judicial afasta o público-alvo (advogados buscando inovação).
*   **Tom Excessivamente Informal:** Usar gírias ou memes em excesso prejudica a credibilidade necessária para tratar de temas jurídicos sérios.
*   **Falta de Call to Action (CTA):** Todo conteúdo deve guiar o leitor para o próximo passo (ler um artigo, testar uma ferramenta, deixar um comentário).

## Verification

Para verificar se o conteúdo está alinhado com a voz da marca, faça um check-list:
1. O texto é fácil de entender para um leigo, mas tecnicamente preciso para um especialista?
2. O conteúdo apresenta uma visão inovadora ou disruptiva sobre o tema?
3. O texto evita o uso desnecessário de jargões?
4. Há uma chamada para ação clara no final?
Se a resposta for "Sim" para todas, o conteúdo está validado.
