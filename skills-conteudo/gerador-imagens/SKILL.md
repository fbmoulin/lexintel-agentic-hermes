# Gerador de Imagens

## When to Use
Use esta skill quando precisar criar imagens de capa (thumbnails), ilustrações para artigos (Pillar) ou assets de fundo para vídeos faceless. A skill não gera a imagem em si, mas o **prompt exato** que deve ser enviado ao gerador de imagens.

## Procedure
1. Receba o tema ou o átomo de conteúdo que precisa de ilustração.
2. Construa um prompt em INGLÊS (melhor performance em geradores) seguindo esta estrutura:
   `[Subject/Concept] + [Environment/Context] + [Lighting/Color Palette] + [Style/Medium] + [Parameters]`
3. **Injeção de Branding Obrigatória:** Sempre inclua na paleta de cores: "dark charcoal background, vibrant neon orange accents, high contrast".
4. **Estilo Obrigatório:** "cyber-legal aesthetic, minimalist, clean lines, UI/UX elements, abstract data visualization, 8k resolution, photorealistic, cinematic lighting".
5. Gere 3 opções de prompt para o usuário escolher.

## Pitfalls
- **NÃO** use clichês jurídicos antigos (balança de latão, martelo de madeira, estátua da justiça de pedra).
- **SUBSTITUA POR:** "holographic scales of justice", "digital glowing gavel", "neural network glowing in orange".
- Nunca inclua texto no prompt da imagem (geradores erram texto). O texto será adicionado depois via tipografia (Montserrat).
