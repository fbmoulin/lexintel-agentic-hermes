# SKILL_FIRAC_PLUS_CIVIL

## Objetivo
Gerar análise FIRAC+ para processos cíveis.

## Estrutura
1. Facts
2. Issues
3. Rules
4. Application
5. Conclusion

## Regras
- Identificar todos os pedidos.
- Identificar todas as defesas.
- Separar preliminares, prejudiciais e mérito.
- Apontar provas relevantes.
- Não decidir tema sem base documental.
