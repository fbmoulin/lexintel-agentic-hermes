# Referências Técnicas

## FastAPI

FastAPI é utilizado como camada de API por permitir construção rápida de endpoints Python com type hints, documentação automática e boa ergonomia para testes.

## Qdrant

Qdrant será utilizado como banco vetorial inicial. A evolução prevista é busca híbrida com vetores densos e esparsos, fusão por RRF/DBSF e posterior reranking.

## OpenAI Agents SDK

A arquitetura de agentes especializados é compatível com conceitos de handoffs, tracing, tool calls e guardrails.

## OWASP LLM

As regras de prompt injection e separação entre conteúdo documental e comandos operacionais seguem recomendações gerais de segurança para aplicações com LLM.

## LangSmith / Avaliação RAG

A lógica de dataset de avaliação, execução comparativa e métricas de qualidade segue o padrão de avaliação contínua de aplicações RAG.
